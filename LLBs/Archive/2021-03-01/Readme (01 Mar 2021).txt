This version of the tool was released on 01 Mar 2021.

It was developed in LV2020CE, and "Saved for previous" (using tool to save LLBs for previous) to LV2018.
The file size in LV2020 is 798,207 bytes (780 KB on disk).

The template files need to be stored in the "\LabVIEW Data\PopupMenus\edit time panel and diagram\Templates" user folder.

Example full path: "C:\Users\Petru\Documents\LabVIEW Data\PopupMenus\edit time panel and diagram\Templates".

The "Templates" path needs to be created by the user.

There are four types of files accepted as templates at the moment: ".vi", ".vit", ".ctl", ".ctt".
